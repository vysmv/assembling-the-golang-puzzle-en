# Изучение Go. Ч.2: документация как инструмент

Привет всем заглянувшим. 

В этом видео я каснусь темы "документация как инструмент". 
Пожалуй что, это важная тема и чем раньше ее поймешь, тем лучше. 

В PHP например документация человекопонятная, очень интуитивная, не формальная, содержит элементы обучающего характера, описывает параметры, типы возвращаемых значений, возможные ошибки примеры применения, компенсирует динамическую дипизацию и имеет специфическую категоризацию.

А вот что мы получаем в распоряжение в экосистеме Go, если откроем [официальный сайт](https://go.dev/)? 

По сути это точка маршрутизации (navigation hub), из которой пользователь перенаправляется к разным типам знаний.
